# RESTAURANT APP

Aplicación para gestionar el menú y las ordenes de un restaurán en tiempo real.

## Tecnologías utilizadas :computer: 

* __React__
* __React Native__
* __Tailwindcss__
* __NativeBase__
* __HOOKS: useContext-useReducer__
* __Firebase__
